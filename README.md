# Student Performance Prediction using Machine Learning

## Introduction
This project aims to predict students’ final academic performance using machine learning techniques based on academic, demographic, and social factors.

## Objective
To build a machine learning model that can predict final student marks (G3) and support early academic intervention.

## SDG Alignment
This project aligns with **SDG 4 – Quality Education** by helping institutions identify students at risk of poor academic performance.

## Tools Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- GitHub
